arsenal_gear.stellar_evolution.isochrone.AbstractIsochrone¶

class arsenal_gear.stellar_evolution.isochrone.AbstractIsochrone(**kwargs)[source]¶

Bases: ABC

Abstract class that represents the mapping from a population’s initial parameters to its properties at a given time. The main instantiaion of this base class is the IsochroneInterpolator class which is used to interpolate isochrones directly from tabulated calculations.
