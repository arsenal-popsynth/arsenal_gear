arsenal-gear¶

A lightweight population synthesis code with an emphasis on the quantities relevant for stellar feedback from massive stars.

Classes

StarMaker([discrete])

This class will act as the primary API for aresenal Ideally it will take an input yaml file or accept default values for certain parameters and return a population of stars with pre-calculated parameters for the feedback and radiaiton.
